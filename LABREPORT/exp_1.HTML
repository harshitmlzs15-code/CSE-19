COURSE CURRICULUM

[ WEB DESIGNING WORKSHOP I : 25VA151 ]

Course Overview

The quick overview of the curriculum is given below:

  1. Introduction to Web Design
  2. HTML Basics
  3. Version Control with Git and GitHub
  4. CSS Fundamentals

  1. Course Objectives

    2. The main objectives of this course are:

    1. Understand the basics of web design.
    2. Learn HTML, Git and GitHub, CSS.
    3. Develop responsive web pages.

  2. Course Structure

    3. The course is divided into theoretical and practical sessions:

    1. Theoretical Lectures
    2. Hands-on Workshops
    3. Project Work

  3. Detailed syllabus

    4. The detailed syllabus for the course is provided below:
